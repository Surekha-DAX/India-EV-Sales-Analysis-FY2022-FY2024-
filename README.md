# ⚡ India EV Sales Analysis (FY2022–FY2024)

📊 **Dataset:** Vahan (2W & 4W)  
🔍 **Focus:** Maker rankings, state penetration, quarterly trends  

## Key Insights
- 🚀 **OLA Electric** leads 2‑wheelers with **322,489 units (FY24)**
- 📈 **BYD India** achieves **567% CAGR (FY22–FY24)**
- 🌍 **Goa** tops 2W penetration at **18%**
- 🌿 **Kerala** leads 4W penetration at **5.8%**
